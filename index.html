Brain Dead game

Player character and gameplay including input, player control, animator setup, weapons, and projectile system.

This twin stick shooter built in Unity gives the player a choice between a female or male player character. All attacks use the projectile system. Melee projectiles have a speed of 0 and only exist for one frame. Projectiles are triggers that hit colliders and activate depending on projectile data and collider layers.

Weapons include fist, pistol, assault rifle, shotgun, serum gun, and alarm clock. The serum gun fires a special projectile that heals injured humans or damages zombies. The alarm clock emits a sound that attracts zombies into a cluster.

Painting with Style

Replication of the research paper "Painterly Rendering with Curved Brush Strokes of Multiple Sizes" by Aaron Hertzmann.

This Python project creates stylized images from photographic images, either automatically or with some manual input from a user. Styles control the appearance of brush strokes such as curvature, min length, max length, and opacity. Each style can have multiple brush sizes of decreasing size to keep refining detail.

Different styles can be used on different layers. An optional masking layer adds detail with smaller brushes only to user-defined areas. These additional features provide the artist more flexibility experimenting with mixing styles.

This is an example of painting a style with 5 decreasing brush sizes. The input is a photo and a style. The canvas begins as white. The canvas is subdivided into a grid based on brush size. In each grid cell, the pixel that is furthest from the correct color in the photo is painted with a circle brush of the correct color.

A separate layer for each brush size is created with the canvas from the previous layer. As the brush sizes decrease, the gaps between larger brushes are filled in.

Sonar Bot

This Arduino differential drive robot uses parallel-mounted ultrasonic sensors to face an object at the surface normal and a proportional-derivative (PD) controller to minimize overshoot and oscillation.

Bitmask Painter

This is a desktop Python app built with tkinter and PyInstaller. A level designer can assign each pixel of a 2D world map to any number of bitmasks by painting the masks. The meaning of each bitmask is determined by the designer. Masks can be added, removed, renamed, or recolored. Games randomly distribute assets by using Poisson disks filtered by the masks before procedurally placing them.
